Quem sou eu

Atuei por quatro anos como pesquisadora de Pós-Graduação analisando dados por meio de técnicas estatísticas e de análise espacial. Neste trabalho, adquiri várias das habilidades necessárias à uma cientista de dados, como pensamento analítico, curiosidade, familiaridade com dados, formulação de perguntas e hipóteses de pesquisa, criatividade, inovação, resolução de problemas e storytelling.

Atualmente trabalho como assessora nivel II em um órgão governamental realizando auditoria de dados de receita.

Além disso, desenvolvo projetos pessoais de Ciência de Dados para aprimorar minhas habilidades de resolução de problemas de negócio aplicando conceitos e ferramentas da área. Busco atuar profissionalmente como Cientista de Dados para gerar valor e uxiliar a tomada de decisão por meio de soluções de dados.

Sou formada em Engenharia Civil e mestra em Transportes pela UFMG, e doutoranda na mesma área na UFPE.

Power BI

Desenvolvimento de Dashboards para empresa de marketplace de aluguel de imóveis com Power BI

Nesse projeto, utilizou-se ferramentas de ETL, conceitos de análise de dados e conhecimento de negócios, associados a ferramentas de DataViz do Power BI (gráficos, cartões, segmentações e mapas), para o desenvolvimento de um relatório contendo dois dashboards para acompanhamento da disponibilidade de imóveis e potencial de receita para uma empresa de marketplace de aluguel de imóveis para curta e longa temporada.

SQL/Power BI

Panorama mundial da Covid-19 por meio de consultas SQL e dashboards no Power BI

Nesse projeto, os conceitos de funções agregadoras, Cláusula CASE, Window functions e, CTE foram utilizados para elaborar consultas SQL para analisar dados mundiais de Covid-19 e obter um panorama do impacto e progressão da doença, bem como dos avanços da vacinação e testagem. Em seguida, os principais insights obtidos foram utilizados para a construção de visualizações no Power BI.


O resultado final do projeto foi: (i) Script SQL contendo a análise completa dos dados, bem como comentários com os insights obtidos; (ii) Relatório no Power BI contendo três dashboards com cartões, gráficos e tabelas apresentando os principais resultados e visualizações para acompanhamento.


O resultado final do projeto pode ser visualizado por meio do link que se encontra no README do projeto no GitHub ou clicando na imagem acima.


Ferramentas utilizadas

  • SQL
  • SQLite
  • Dbeaver
  • Power BI
  • GitHub
  • Portfólio GitHub
